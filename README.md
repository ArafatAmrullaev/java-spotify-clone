# Spotify Clone Project

## Description
This project is a Spotify clone developed using the Spring Framework. It aims to replicate some of the core functionalities of the Spotify music streaming service.

## Features
- Browse artists, albums, songs, and playlists
- Create and manage playlists
- Search for music(in progress)
- Play music with streaming capabilities(in progress)

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- Spring Security
- H2 Database (for development)
- PostgreSQL (for production)
- MapStruct (for object mapping)
- Swagger UI (for API documentation)

## Getting Started
Follow these instructions to get the project up and running on your local machine.

### Prerequisites
- Java 21 or higher
- Maven
- IDE (IntelliJ IDEA, Eclipse, etc.)
